# FightingGames2D

## 概要

Unity6で制作した2D格闘ゲームです。
キーボードおよびコントローラー操作に対応し、プレイヤーとAIによる対戦形式でのバトルを実装しています。

---

## 動作環境
* PC（Windows）
* Unity6
* キーボード操作対応
* Xbox系コントローラー操作対応
※ コントローラーが無くてもプレイ可能です

---

## 利用ツール
* Unity6
* C#
* Visual Studio
* GitHub

---

## 制作体制
個人制作
企画・設計・プログラム実装を一貫して担当しました。

## 操作方法
### 移動

* キーボード：W / A / S / D
* コントローラー：左スティック

---

### 攻撃

* LP：Yキー / コントローラー（Xボタン）
* MP：Uキー / コントローラー（Yボタン）
* HP：Iキー / コントローラー（RB）
* LK：Hキー / コントローラー（Aボタン）
* MK：Jキー / コントローラー（Bボタン）
* HK：Kキー / コントローラー（RT）

---

### ガード（シールド）

* キーボード：Left Shift または Space
* コントローラー：LT / LB

※ 相手と反対方向に入力した場合もガードが成立します

---

## 工夫した点

* 攻撃をフレーム単位で管理し、発生・持続・硬直を制御
* 当たり判定（ヒットボックス / ハートボックス）を用いた衝突判定の実装
* AIキャラクターの行動制御（距離に応じた行動選択など）
* プレイヤーの向きに応じて、逆方向入力でガードが成立する仕様を実装
